Edit `index.html` and change:

**Your Name:**
```html

```

**Your Email:**
```html

```

**Your Location:**
```html

```

## 🐛 Troubleshooting

### Issue: Website won't open
**Solution**: Right-click `index.html` → Open with → Browser

### Issue: Styles aren't loading
**Solution**: Make sure all files are in the same folder:
```
portfolio/
├── index.html
├── styles.css
├── script.js
└── resume.pdf
```

### Issue: Theme toggle not working
**Solution**: Clear browser cache (Ctrl+Shift+Delete)

### Issue: Dark mode not saving
**Solution**: Check if localStorage is enabled in browser

### Issue: Icons not showing
**Solution**: Check internet connection (Font Awesome uses CDN)
